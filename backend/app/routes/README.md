# API Routes

This module contains all API endpoints for the blog system.

## Routes Included

- **blogs.py**: Blog-related endpoints
  - GET /blogs - List all blogs
  - GET /blogs/{slug} - Get single blog
  - GET /blogs/search - Search blogs
  - GET /blogs/tags - Get all tags
  - GET /blogs/featured - Get featured posts
  - GET /blogs/related/{slug} - Get related posts
